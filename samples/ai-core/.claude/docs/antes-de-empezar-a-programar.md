# Antes de Comenzar a Programar

> **URL oficial:** https://facturascripts.com/antes-de-empezar-a-programar-580

Es fundamental que no realices cambios en los archivos de las carpetas **Core** o **Dinamic**, ya que cualquier modificación se perderá al actualizar el sistema. **Todas las personalizaciones deben implementarse a través de plugins**; por lo tanto, [crea un plugin](/publicaciones/creacion-de-plugins-210) y coloca tus adaptaciones allí.

## 💬 Chat para Programadores
Disponemos de un chat en **Discord** para los programadores de FacturaScripts, así como un roadmap. Además, llevamos a cabo reuniones por videoconferencia todos los martes para resolver dudas y coordinar trabajos. Para más información, visita el [grupo de desarrollo](/colabora/DEVELOPMENT).

![Comunidad Discord](/MyFiles/2023/11/1790.png?myft=930c1d47c0c17ae188ffa3c41c7bbcc96f5dc244)

## ⚠️ Notificación de Errores
Utiliza la [sección de contacto](/contacto) para informar sobre errores. Si el problema está relacionado con un plugin, selecciona el plugin correspondiente para que la incidencia sea notificada al autor. Cuando el autor responda, recibirás una notificación por correo electrónico. Esta es la forma más efectiva de resolver cualquier problema y no quedarte atrás.

![Contactar Autor del Plugin](https://i.imgur.com/XlK4V7I.png)

## 💻 Lenguaje de Programación
FacturaScripts está desarrollado en PHP, por lo que necesitarás conocimientos básicos de este lenguaje para programar. Si aún no tienes experiencia con PHP, puedes realizar nuestro [curso de programación en PHP](/cursos/curso-de-php).

### 📚 Documentación de Clases
Para conocer las propiedades y métodos de cada clase, disponemos de documentación generada a partir del código fuente en [doc.facturascripts.com](https://doc.facturascripts.com/).

### 📊 Esquema de la Base de Datos
Encontrarás el [esquema de columnas y restricciones de cada tabla](/publicaciones/diagramas-de-tablas) en los directorios [Core/Table](https://github.com/NeoRazorX/facturascripts/tree/master/Core/Table) y **Dinamic/Table**. Estos son los archivos que utiliza FacturaScripts para crear y actualizar la estructura de las tablas.

### ✍️ Editor de Código
Un buen editor de código puede hacer que el proceso de programación sea más cómodo. Los editores más utilizados para el desarrollo de FacturaScripts son [phpStorm](https://www.jetbrains.com/phpstorm/) y [Visual Studio Code](https://code.visualstudio.com/).

#### 💾 Git / GitHub
Aunque no es obligatorio, se recomienda encarecidamente que te familiarices con Git:
- [Guía de manejo de Git](https://rogerdudler.github.io/git-guide/index.es.html)
- [Cómo hacer un pull request](https://www.freecodecamp.org/espanol/news/como-hacer-tu-primer-pull-request-en-github/)
