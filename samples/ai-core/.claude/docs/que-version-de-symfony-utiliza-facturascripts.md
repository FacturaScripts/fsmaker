# ¿Qué versión de Symfony utiliza FacturaScripts?

> **ID:** 610 | **Permalink:** que-version-de-symfony-utiliza-facturascripts-539 | **Última modificación:** 26-08-2025
> **URL oficial:** https://facturascripts.com/que-version-de-symfony-utiliza-facturascripts-539

La respuesta breve es **ninguna**. FacturaScripts no se basa en el núcleo de Symfony, ya que hemos desarrollado nuestro propio núcleo para abordar necesidades diferentes a las que resuelve Symfony.

## ⚙️ Componentes de Symfony utilizados

No obstante, utilizamos algunos componentes de Symfony para mejorar nuestra funcionalidad, tales como:

- **Twig**: Un motor de plantillas que facilita la creación de vistas.

Es importante destacar que no es necesario consultar la documentación de Symfony para desarrollar en FacturaScripts, ya que toda la información necesaria está disponible en esta misma web.

### 🔄 Anteriormente
Hasta la versión 2025 también usábamos los componentes **HttpFoundation** y **Translator** de Symfony.
