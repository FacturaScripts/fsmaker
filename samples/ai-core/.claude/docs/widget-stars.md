# Widget Stars

> **ID:** 1628 | **Permalink:** widget-stars | **Última modificación:** 31-10-2025
> **URL oficial:** https://facturascripts.com/widget-stars

El widget de estrellas, o **WidgetStars**, permite seleccionar un calificación entre 0 y 5 estrellas. Este widget es perfecto para calificaciones de productos, servicios, etc.

```
<column name="rating" numcolumns="3" order="100">
	<widget type="stars" fieldname="rating"/>
</column>
```
